What is the Document Object Model (DOM)?

Document Object Model (DOM) is a structural blueprint of a web page. It construct tree like stature of web page to understand it's foundation.

How do you select an element from the DOM?

There are several ways to select an element from the DOM:

Finding HTML elements by id > document.getElementById("id");
Finding HTML elements by tag name > document.getElementsByTagName("p");
Finding HTML elements by class name > document.getElementsByClassName("intro");
Finding HTML elements by CSS selectors > document.querySelectorAll("p.intro");
Finding HTML elements by HTML object collection > document.forms["frm1"];

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.
